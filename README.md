# Northwind cli tool

Dependences to install

### Nodejs and Npm
```
https://nodejs.org/es/download/package-manager/
```

### Install Commander
```
npm install commander --save
```

### Install file system
```
npm install file-system --save
```

### Install extra file system
```
npm install --save fs-extra
```

### Install path module
```
npm install --save path
```
